# Weather in Australia Demo

Software Requirement:
* jdk 1.8
* springboot 2.1.1.RELEASE
* maven 3.0+

After run project(springboot-jsp-weather) in your IDE,   
or mvn clean package and java -jar ...  
See http://localhost:5000 in your browser.
